# Contact Orchestration Pipeline

## Arquivos de entrada (Inputs)

O pipeline utiliza os seguintes arquivos como fonte de dados:

- complicacao.xlsx
- internacao.xlsx
- status.csv
- status_resposta_complicacao.csv
- status_resposta_internacao.csv
- status_resposta_eletivo.csv

Todos os arquivos devem estar localizados no diretório de execução do pipeline e nos seus devidos formatos.
